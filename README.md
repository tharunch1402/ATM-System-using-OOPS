# ATM PIN System using Python (OOPS)
This is a beginner-level ATM PIN system created using Python to practice Object-Oriented Programming (OOPS) basics.

## What this project does
- Takes ATM PIN input
- Shows menu options after login
- Check balance
- Deposit money
- Withdraw money
- Change PIN option

## Concepts used
- Classes and Objects
- Basic Encapsulation
- Menu-driven program logic

This project is part of my learning journey and created for practice purposes.
